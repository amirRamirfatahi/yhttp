route(pattern='/', verb=None)¶

Decorator factory to register a handler for given regex pattern. if verb is None then the function name will used instead.

@app.route(r'/.*')
def get(req):
    ...

Regular expression groups will be capture and dispatched as the positional arguments of the handler afet req:

@app.route(r'/(\d+)/(\w*)')
def get(req, id, name):
    ...

This method returns a decorator for handler fucntions. So, you can use it like:

books = app.route(s'/books/(.*)')

@books
def get(req, id):
    ...

@books
def post(req, id):
    ...
Parameters

pattern – Regular expression to match the requests.
